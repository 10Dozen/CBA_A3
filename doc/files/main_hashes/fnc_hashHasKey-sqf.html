CBA_fnc_hashHasKey

Description

Check if a Hash has a value defined for a key.

Parameters

_hashHash to look for key in [Array which is a Hash structure]
_keyKey to search for in Hash [Any]

Returns

True if key defined, false if not defined [Boolean]
